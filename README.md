# Portfolio Studente

Benvenuti nel repository del mio progetto di portfolio personale. Questo è un progetto didattico creato per esercitarsi con le basi dello sviluppo web (HTML5 e CSS3) e l'utilizzo del sistema di controllo versione Git.

## Scopo del Progetto

L'obiettivo di questo progetto è creare una pagina web statica che presenti il mio profilo, le mie competenze e i progetti realizzati, imparando a gestire il flusso di lavoro professionale attraverso commit, branch e pubblicazione su GitHub.

## Struttura del Progetto

Il repository è organizzato come segue:

- `index.html`: Struttura principale della pagina e contenuti.
- `style.css`: Foglio di stile per la veste grafica (colori, font e layout).
- `README.md`: Documentazione del progetto (questo file).

## Tecnologie Utilizzate

- **HTML5**: Per la marcatura dei contenuti.
- **CSS3**: Per la personalizzazione estetica.
- **Git**: Per il controllo di versione.
